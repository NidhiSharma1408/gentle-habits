## Project

- This is a client-side React SPA using Vite, Zustand, Framer Motion, and CSS Modules.
- No backend — all data is in localStorage.
- The Claude API integration uses a Vite dev proxy (`/api/claude` -> `api.anthropic.com`) to avoid CORS.
